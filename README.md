## 服务单随机生成系统（PDF填表）

>  软件主要功能就是自动生成服务数据，然后填写PDF模板，并导出到指定目录，涉及到的主要知识点有以下内容：
文本分割
数组操作
正则表达式
listview读取写入
PDF填表操作
PDF导出操作


**主要关键知识点：**

PDF表单制作（模板）
PDF填表操作（ITEXTCSHARP）
